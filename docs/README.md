## Mise en ligne

Dans GitHub :

1. pousse le depot
2. va dans `Settings > Pages`
3. choisis `Deploy from a branch`
4. selectionne la branche voulue
5. selectionne le dossier `/docs`

GitHub Pages servira ensuite directement [index.html](/Users/tael/Documents/FireTechDev/UDSP31/docs/index.html).
