# Theme `udsp31`

Theme WordPress classique pour la home page UDSP31.

## Installation

1. Copier le dossier `udsp31` dans `wp-content/themes/`.
2. Activer le theme dans l'administration WordPress.
3. Definir une page statique comme page d'accueil si besoin.
4. Assigner les menus `primary`, `footer` et `legal`.
5. Renseigner les coordonnees et le hero dans `Apparence > Personnaliser > UDSP31`.

## Contenus relies a WordPress

- La section `Actualites` affiche les 3 derniers articles.
- Les logos partenaires sont pour l'instant des cartes texte.
- Le logo principal peut etre remplace via `Identite du site`.
